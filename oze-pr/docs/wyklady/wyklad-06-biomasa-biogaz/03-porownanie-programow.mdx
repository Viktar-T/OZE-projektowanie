<Slide title="🔧 Przegląd narzędzi" type="info">

<KeyPoints title="⚗️ Narzędzia procesowe (CAE)">
- 🔬 **SIMBA#**: Elastyczne flowsheety, integracja sterowania, narzędzia kalibracji
- 🚀 **BioWin**: Przyjazny interfejs, gotowe jednostki (digestery, CHP), raporty
- 🎯 **GPS‑X**: Duża konfigurowalność, identyfikacja parametrów
- 🌊 **WEST**: Modułowość, integracje hydrodynamiczne
</KeyPoints>

<SupportingDetails title="💰 Narzędzia ekonomiczne i optymalizacyjne">
- 📊 **RETScreen**: Szybkie LCOE/NPV, biblioteki kosztów, scenariusze
- ⚡ **HOMER Pro**: Profilowanie obciążeń, dispatch, magazyny, optymalizacja miksu
- 🔗 **Integracja**: Eksport CSV/XLSX, API/skrypty, SCADA (OPC UA)
</SupportingDetails>

<KeyPoints title="📊 Poziom szczegółowości">
- 🔬 **Dynamika procesu** (VFA, pH) → SIMBA#/BioWin/GPS‑X/WEST
- 💰 **Makro‑ekonomia** → RETScreen
- ⚡ **Miks/sterowanie energią** → HOMER Pro
</KeyPoints>

<SupportingDetails title="🔗 Integracje i koszty">
- 📁 **Integracje**: eksport CSV/XLSX, API/skrypty, SCADA (OPC UA) – ważne w projektach badawczych
- 💰 **Licencje i koszty**: uczelniane vs. komercyjne; moduły dodatkowe (CHP, upgrading, odory)
- 📚 **Krzywa nauki**: szybki start – BioWin; pełna kontrola i sterowanie – SIMBA#/WEST/GPS‑X
</SupportingDetails>

<Slide title="🔄 Workflow między narzędziami" type="success">

<KeyPoints title="📈 Sekwencja pracy">
- ⚗️ **1. Model procesu** (SIMBA#/BioWin) → budowa schematu, parametryzacja
- 📊 **2. Profil produkcji** gazu/energii (CSV) → eksport wyników
- ⚡ **3. Optymalizacja miksu** i magazynowania (HOMER Pro) → strategia energetyczna
- 💰 **4. TEA/finanse** (RETScreen) → analiza ekonomiczna
- 📝 **5. Raport** → podsumowanie i rekomendacje
</KeyPoints>

<SupportingDetails title="🔗 Kluczowe interfejsy">
- 📁 **Formaty wymiany**: CSV, XLSX, JSON
- 🔄 **Iteracje**: Wyniki z jednego narzędzia → dane wejściowe do następnego
- 📊 **Walidacja**: Sprawdzenie spójności między narzędziami
- 🎯 **Optymalizacja**: Iteracyjne doskonalenie parametrów
</SupportingDetails>
