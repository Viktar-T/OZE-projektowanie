## Procedura

1. Import profili aerodynamicznych (XFoil/NREL) i definicja łopaty.
2. Analiza BEM – charakterystyka mocy, momentu, obciążeń.
3. Symulacje niestacjonarne – gusty IEC, manewry start/stop.
4. Eksport krzywych mocy do TOPFARM/WindPRO.

## Wskazówki

- Monitorować zbieżność obliczeń (liczba pierścieni, kryteria convergence).
- Aktualizować dane turbulencji zgodnie z klasą IEC.
