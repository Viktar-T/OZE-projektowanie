title: "Sunny Design — schemat procesu projektowego"
---

1. Szybkie projektowanie: lokalizacja (mapa), parametry bazowe, automatyczny dobór komponentów SMA, wstępne wyniki.
2. Planowanie wizualne: kontur dachu (rysunek/import), automatyczne rozmieszczenie modułów, analiza zacienienia 3D, optymalizacja stringów.
3. Konfiguracja szczegółowa: dobór falowników i modułów, okablowanie, ustawienia magazynu energii.
4. Symulacja energetyczna: produkcja roczna, autokonsumpcja, praca magazynu.
5. Analiza ekonomiczna: taryfy lokalne, kalkulacje finansowe, prognozy rentowności.
6. Dokumentacja: raporty, eksport schematów, materiały dla klienta.
