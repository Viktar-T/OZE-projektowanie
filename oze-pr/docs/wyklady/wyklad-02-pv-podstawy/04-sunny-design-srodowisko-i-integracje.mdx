Sunny Design jest zoptymalizowany do szybkiego, iteracyjnego planowania instalacji, szczególnie w ekosystemie SMA.

## Środowisko projektowe
- Tryb szybkiego projektowania (kilka kroków) i tryb szczegółowy.
- Planowanie 2D/3D, import zdjęć i modeli.

## Funkcje specjalizowane
- Współpraca zespołowa, role i uprawnienia, personalizacja dokumentów.
- Analiza techniczna: ładowanie EV z optymalizacją PV, import szeregów cen dynamicznych, analiza P‑Q dla magazynów.

## Integracje SMA
- Eksport konfiguracji do SMA Data Manager przyspiesza uruchomienie systemu.
- Automatyczne propozycje urządzeń SMA optymalnych dla projektu.
