## PV*SOL — kluczowe operacje
- File → New Project: wybór typu systemu i źródeł danych klimatycznych.
- PV Modules: definicja powierzchni i układu, wizualizacja 2D/3D, zacienienie.
- Inverters: automatyczna konfiguracja, kontrola zakresów MPPT.
- Simulation: uruchomienie obliczeń, monitoring postępu.
- Results: analiza wyników, eksport danych i raportów.

## Sunny Design — workflow praktyczny
- Rejestracja/logowanie, New Project i wybór typu instalacji.
- Location Selection: mapa z danymi klimatycznymi.
- Visual Planning: obrys dachu (rysunek/import), rozmieszczenie modułów.
- Component Selection: propozycje SMA, System Configuration: weryfikacja/optymalizacja.
- Results & Documentation: raporty i dokumentacja techniczna.
